* Spark2.1.1安裝
* Scala2.12.2安裝
* python2.7.13安裝程序
  + 有安裝anaconda就可以不用裝
* anaconda安裝
  + 推薦直接安裝anaconda，自帶jupyter notebook及各式常用模組
* FAQ
  + 安裝過程中遇到問題解法
